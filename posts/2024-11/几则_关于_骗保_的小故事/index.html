几则，关于“骗保”的小故事 by 循因缉药

世界是多样的。


每人都有自己的观察立场和自洽逻辑。有的遵从法理，有的诉诸人情。


世界又是如此的割裂。


舆论场的常态：先选边站队，然后骂对面sb。


比如，医疗圈这几天讨论最热烈的医保骗保事件。


但今天，我们聊点其他的。


几则小故事，根据平行世界现实案例演绎，请勿对号入座。


  • 故事1：

老徐是安徽某县一农民，收入不高，但日子过得也算平淡幸福。然而突发噩耗，老徐的妻子被诊断出了癌症。为了给妻子治病，老徐花光了家里所有的积蓄，甚至负债累累。


这天，老徐偶然打听到有一种靶向药对这一肿瘤有奇效。如果自费，每个月得花好几万；好消息是这个药可以医保报销，报销后每个月只需要几千元。但医保报销的前提条件是，需要做基因检测，且报告中A基因为阳性才可以报销。


这个检测，医院没办法自己做。在医生的介绍下，老徐找到了X基因检测公司代表张三。并通过张三给妻子做了基因检测，结果A基因为阴性，不满足报销条件。


为了能吃上医保药，老徐恳请张三想想办法。张三觉得老徐一家确实可怜，并且国外也有A基因阴性但吃该靶向药仍有效的案例，于是，张三用软件将检测报告改为阳性。


老徐拿着被修改的检测报告，顺利买到了靶向药，并进行了医保报销。



  • 故事2：


老徐是安徽某县一农民 ... 有奇效 ... 但需要检测报告 ...


老徐通过检测公司代表张三，给妻子做了基因检测，但结果A基因为阴性。


为了能吃上医保药，张三暗示老徐可以自己修改报告。老徐找了一家打印店，将检测报告改为阳性。


老徐拿着被修改的检测报告，顺利买到了靶向药，并进行了医保报销。



  • 故事3：


老徐是安徽某县一农民 ... 有奇效 ... 但需要检测报告 ...


X基因检测公司代表张三，为了完成业绩，谎称可以帮助老徐拿到阳性报告。老徐花费万元在张三处做了检测后，张三在明知结果为阴性、吃药无效的情况下，瞒着老徐，将检测报告改为阳性。


老徐拿着被修改的检测报告，顺利买到了靶向药，并进行了医保报销。



  • 故事4：


老徐是安徽某县一农民 ... 有奇效 ... 但需要检测报告 ...


X基因检测公司代表张三，为了完成业绩，谎称可以帮助老徐拿到阳性报告。


在收到老徐近万元的检测费用后，张三并未将样本送至公司检测，而是用软件直接杜撰了一份假的阳性报告。


老徐拿着该检测报告，顺利买到了靶向药，并进行了医保报销。



  • 故事5：


老徐是安徽某县一农民 ... 有奇效 ... 但需要检测报告 ...


在医生的介绍下，老徐找到了本地区负责该靶向药的Y药企代表李四，并在李四的推荐下，联系了X基因检测公司代表张三，通过张三做了基因检测。


在明知检测报告为阴性的情况下，为了顺利卖药完成业绩，李四将检测报告修改为阳性。


老徐拿着被修改的检测报告，顺利买到了靶向药，并进行了医保报销。



  • 故事6：

老徐是安徽某县一农民 ... 有奇效 ... 但需要检测报告 ... 张三 ... 李四 ...


李四通过这种方式，顺利拿到了地区销冠，得到Y公司嘉奖。


Y公司领导在明知该方式违法的情况下，仍然默许、暗示、鼓励、威压其他销售代表学习这种方式，并提高业绩指标，给代表们施加压力。

  • 故事7：


为了扩大销售规模、避免患者流失，Y药企与X检测公司达成合作，Y药企一次性采购X检测公司X检测5万份。在*肿瘤患者中，Y药企免费提供X检测服务，对X检测阳性的患者，给予Y药企靶向药治疗，同时进行医保报销。


一段时间后，Y药企负责人为提升业绩，暗示X检测公司可以适当提高检测阳性率，让更多的肿瘤患者具备医保购药资格，这样，Y药企也能卖更多的药。


X检测公司于是随机、按Y要求指定对某些报告进行修改....


老徐在毫不知情的前提下，拿到了被修改的检测报告，顺利买到了靶向药，并进行了医保报销。

但最终，也没能留住妻子的生命。


为了还债，老徐想起了前段时间在医院门口收药的药贩子，于是把家里剩余的药，都卖给了药贩子；在药贩子的“建议”下，老徐又铤而走险，用刚去世妻子的医保卡，在县医院又开了十多盒靶向药，再次卖给了药贩子。


药贩子随后以低于市场价30%的价格，将该药卖给了患者老王。

故事9、故事10


.....


所以，即使在“篡改基因检测报告 → 骗保”违法事实已经坐实的背景下，每一个个案，可能还会存在多种细节，影响大家的观感，甚至法院的判决。


  • 患者对修改检测报告一事是否知情？是否主动要求修改报告？

  • 销售代表是否主动or被动修改报告？动机是什么？是否从中获利？

  • 是个人行为，还是公司行为？

  • 企业对这一情况是否知情？有无默许、暗示、明示该情况发生？有无获利？

  • 企业是否成体系、成建制开展类似活动？

  • 涉案金额大小？

  • 是否有证据支持该药对于该类型患者有效？这种情况的发生，是否与医保纳入该药适应征的滞后性有关？

  • 从体系上，为什么，报告会先流转到代表的手中？

  • 医生是否知情，医生是否因利有不当推荐？

  • ...


真实世界，这些故事，正在发生。


真实世界，远比非黑即白，更复杂。


唯一需要注意的是：


如果你是围观者，不要轻易被各种信息带节奏；


如果你是局中人，即使你可以找到道德上的自洽，但也不要，把自己推到法律上万劫不复的境地。
