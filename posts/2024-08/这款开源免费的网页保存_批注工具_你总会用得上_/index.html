01 引言

遇到好的网页/文章，当下没时间阅读，我们都希望将它保存下来，以后有时间再看！也就是所谓的稍后读。

基于此需求，一大堆稍后读 app 诞生了，我就分享过很多款（真是能折腾😅）:

  • 简悦: 最简单的【简悦】一键剪藏网页全文至【Obsidian】的方案（无需简悦高级账户）

  • Omnivore: 全平台开源免费稍后读利器，且可与Obsidian联用！

  • Cubox: Obsidian的最强搭档

  • Circle: 一款免费、配置简单、功能强大的浏览器阅读助手

  • 会读: 如何10 倍提升你的阅读效率

  • Pigenote: 永久可用的【网页划线标注】利器，支持导出Markdown

  • ......

Info

我上面列的只是一小部分，其实还有很多。

不仅工具很多，保存网页的格式也有很多，关注我的小伙伴可能都希望保存成 Markdown 格式并导入 Obsidian🤣，这也是很多稍后读 app 都支持的一种格式。

但上面这种方式有两个不好的点：

  • 一部分配置有些麻烦

  • 收藏多了，却没时间阅读，时间长了会“污染” Obsidian 知识库

另外，有一些小伙伴希望将网页的原样式也一起保存下来，也就是说保存成 Html 格式。如果保存时还能批注做笔记就更好了！

Tip

可能你会说直接右键另存为 html 格式就是完事了，但是你会发现你保存的可不仅仅是一个 html 文件，还附带一个文件夹，里面包含各种图片、css 等，非常不利于管理、分享和多端同步阅读，并且还有可能加载失败，更没法批注。

针对上述这些问题与需求，今天我就给各位小伙伴分享一个开源免费的利器，轻松搞定~

02 SingleFile

这个利器的名字就叫 SingleFile，它是一款浏览器插件，支持以下浏览器：

  • Firefox: https://addons.mozilla.org/firefox/addon/single-file

  • Firefox for Android: https://addons.mozilla.org/android/addon/single-file

  • Chrome: https://chrome.google.com/extensions/detail/mpiodijhokgodhhofbcjdecpffjipkle

  • Safari (macOS and iOS): https://apps.apple.com/us/app/singlefile-for-safari/id6444322545

  • Microsoft Edge: https://microsoftedge.microsoft.com/addons/detail/efnbkdcfmcmnhlkaijjjmhjjgladedno

它的拿手绝活就是将整个网页保存为单个 Html 文件，文本、图片、图标以及其他各种元素都会嵌入在里面，十分便携，并且在各端都能打开与查阅。

但是它的强大远不止于此......

下面我一一道来！

2.1 功能与使用

保存全文

这个很简单。

安装好插件之后，直接单击 SingleFile 插件图标，即可实现网页原样式保存。

保存所选内容

并不是每个人都喜欢保存全文，并且为了避免信息泛滥，在《打造第二大脑》这本书里给出了一个有效对策：

仅记录下震撼人心的重磅信息并保存在可靠的位置，对于其他内容能省则省即可。

Tip

所谓的重磅信息，能够直击你的内心深处。它们往往是最与众不同的构思、最颠覆常规的异想、最妙趣横生的设计，抑或最富于潜力的创意。

这个功能的使用方法: 需要你先选中网页上想保存的内容，然后在扩展工具栏或网页上右击 SingleFile 按钮以打开上下文菜单

保存标签页

这个主要适用于批量保存的情况，如果当前你已经打开了多个网页（标签页），那么：

  • 你可以选定几个标签页来保存（按住 Ctrl 键，然后点选目标网页即可）

  • 也可以保存未固定的标签页 （Edge 浏览器是支持固定标签页的）

  • 也可以一键保存所有标签页

使用方法：同样是点击插件工具栏图标或者网页空白处，然后右键菜单

自动保存

可以自动保存当前、为固定&所有标签页。

这个我是不建议使用的。没必要~

网页批注

Tip

也许很多小伙伴之前用过 SingleFile，但只会用它来保存网页，并不知道它也是可以用来做网页批注的。

SingleFile 不仅可以做网页批注，而且批注工具还有很多种类，请见下图

做好批注，点击最右端的保存图标即可，将标注过的 html 文件分享给别人，一目了然！

使用方法：右键网页或者插件图标，选择标注并保存该页面。

保存所选链接

这个有点儿强大啊，很多小 shou 伙 cang 伴 jia 应该都需要。

以我之前的 一篇推文为例 🍉，在文章末尾有一堆精彩推荐

你只要选中上面这些链接，然后右键 SingleFile→保存所选链接，即可批量提取出网址并保存

2.2 一点儿配置

打开插件选项，进入配置界面

乍一看，配置项还挺多的，不过请放心，不是每一条配置都需要去配置的。

甚至直接使用默认配置都是可以的。下面是我推荐的几个配置项

  • 自定义文件名模板

  • 保存位置。支持保存到本地电脑、剪贴板、github、webdav 等

其他的请按需配置吧。

One More Thing

我不知道你们有没有这个困扰：就是保存了很多稍后读，然后就再也没读过了！

针对这个问题，一共有两个小技巧，本次先分享一个与 SingleFile 有关的小技巧。

我们在保存网页时，利用 SingleFile 的标注功能，添加一个简单的批注——这个网页中的内容它能在当下或者未来能在什么方面帮助我，也就是保存这篇文章的目的是什么。

Tip

也许你不用 SingleFile 进行网页保存，那其实这个方法也实用于其他稍后读工具。

这样标注之后，才会促进你思考，更有助于你进行有效筛选，同时当你以后打开这篇文章时，也能通过之前标注很快知道这篇文章对我会有什么帮助（即使未来的看法与当下的想法不一致）。
