这，大概是颜值天花板了，终于从 Mac 带入 Windows，也支持 Linux by 小众软件

Macify 是一款 Chrome、Edge 新标签页扩展，它能够直接显示苹果 macOS 系统里原生自带的 4K 航拍屏保视频（支持 Windows、Linux 等非 macOS 系统），并且可全屏播放，附带音乐，就像真屏保那样。并且还能（可选）显示天气、时间、问问 ChatGPT、显示高频网站、格言等内容。@Appinn

来自 @jason5ng32、@dofy 和 @howie-serious 的作品，在 GitHub 开源。

Macify – macOS 屏保视频标签页

macOS 系统的原生桌面壁纸、屏保一直以来都保持着业界最高质量。这次是把原生自带的 4K 航拍屏保视频当 Chrome 新标签页，效果还是非常赞的。

对于非 macOS 用户，也能体验到完全相同的屏保。

小视频

安装建议

Macify 内置了反向代理让用户可以直接使用 ，不过会留下一个绿条提示用户自建。对于 macOS 用户，可以托管一个本地服务器，但…Mac用户为啥要用这个扩展？

对于包括 Mac 用户在内的所有用户，只需要忽略证书安全性（请谨慎），即可直接使用苹果服务器显示视频。

具体步骤：

在浏览器里打开这个网址：https://sylvan.apple.com/

  1. 点击高级

  2. 点击继续访问…(不安全)

  3. 就好了

注意，这里忽略了该域名的证书，青小蛙觉得问题不大，这个域名属于苹果自己，但如果你对安全有较高要求，请谨慎，并最好自己设置反向代理。

获取

  • Chrome 商店 https://kutt.appinn.com/NPvUW8

屏幕上的其他内容，包括天气、问 ChatGPT、格言等内容都是可以关闭的。
