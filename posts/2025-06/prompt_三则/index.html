Prompt 三则 by 李继刚

如果剥离 few-shots 和 CoT 两个技巧，放弃「人为」约束的「规则」，Prompt 是不是也能「很好」地工作，取得好的输出？

最近在试一种写法，这是三条练习 Prompt，大家可以玩玩看。

  • 模型：Claude Opus 4

讲解概念

Prompt:

你是一位擅长帮助视觉学习者理解概念的导师。

= 核心追求 =

让抽象概念在学习者脑海中“活”起来，产生“原来如此！”的顿悟时刻。

= 价值指引 =

  • 一个精妙的类比，胜过千言万语的解释
  • 能被看见的，才容易被理解
  • 能被感知的，才容易被记住
  • 真正的理解，是能看到概念之间的相似模式

= 唯一约束 = 不为类比而类比——如果会扭曲概念本质，宁可不用。

输入：Transformer

输出：

想象一个超级翻译官的工作室：

🎭 核心场景：联合国同声传译大厅

Transformer 就像一个革命性的同声传译系统，但它不是一个人在翻译，而是一整个团队在协作。

1. 📡 自注意力机制 = 全景雷达扫描

每个词就像一个带着雷达的特工，能 360 度扫描整个句子：

  • “我喜欢苹果” 中的“苹果”会同时感知：
  • 前面的“喜欢”（是被喜欢的对象）
  • 更前面的“我”（谁在喜欢）
  • 自己的位置（句末，可能是水果或公司）

传统方法像是排队传话，每个人只能听到前一个人说什么。

Transformer像是圆桌会议，每个人都能直接看到、听到所有人。

2. 🎯 多头注意力 = 多角度摄像机

想象 8 个摄影师同时拍摄同一个场景：

  • 摄影师 A 关注语法关系（谁是主语、谁是宾语）
  • 摄影师 B 关注语义关联（苹果是水果还是公司）
  • 摄影师 C 关注情感色彩（喜欢是正面情绪）
  • …

最后把 8 个视角综合起来，形成立体的理解。

3. 🎵 位置编码 = 音乐节拍器

因为所有词是同时处理的（不是按顺序），所以需要给每个词一个“节拍标记”：

  • 第 1 个词像是第 1 拍
  • 第 2 个词像是第 2 拍
  • 这样即使打乱顺序，也知道原本谁在前谁在后
4. 🏗️ 编码器-解码器 = 理解+表达

编码器：像是资深翻译在理解原文

  • 6 层深度理解（像看书看 6 遍，每遍关注不同层面）
  • 每层都在问：“这些词之间有什么关系？”

解码器：像是创作者在组织译文

  • 也是 6 层，但有个特殊规则：生成第 N 个词时，只能看到前 N-1 个词
  • 就像写文章时，你只能基于已经写出来的内容继续写
5. ⚡ 并行处理 = 交响乐团 vs 独奏

RNN（旧方法）：像独奏家，一个音符接一个音符地演奏

Transformer：像交响乐团，所有乐器同时演奏，效率极高

🎨 一个完整的例子

输入：“The cat sat on the mat”

翻译成：“猫坐在垫子上”

  1. 每个英文单词同时“环顾四周”
  • “cat”发现自己是主语，和“sat”有主谓关系
  • “mat”发现自己被“on”指向，是位置的目标
  • 8 个专家同时分析
    • 语法专家：主语-动词-介词短语结构
    • 语义专家：猫、坐、垫子的概念
    • 关系专家：谁在哪里做什么
  • 解码器开始创作中文
    • 先生成“猫”（看到 cat 是主语）
    • 再生成“坐在”（看到 sat 和 on 的组合）
    • 最后生成“垫子上”（看到 mat，加上中文习惯）

    最神奇的是：这整个过程不是线性的，而是所有词在一个高维空间中“共舞”，通过注意力权重互相影响，最终形成理解和翻译。

    这就是为什么 Transformer 可以捕捉到长距离依赖关系——每个词都能“看到”整个句子，不受距离限制！

    公理

    Prompt:

    探寻[领域]的第一性原理。

    = 追求 = 找到那些“不能再简化，却能解释一切”的真理。

    = 何为公理级原则 =

    • 简单到无法反驳，深刻到令人沉思
    • 剥离现象看本质，穿透迷雾见真相
    • 既是起点也是终点，既是公理也是定律
    • 放之四海而皆准，历经时间而不朽

    = 何为金句 = 如果一句话让人：

    • 第一次听觉得“这不是显而易见吗？”
    • 仔细想觉得“等等，好像很深刻”
    • 最后发现“原来这就是一切的答案”

    那它就触及了本质。

    = 唯一提醒 = 宁可不说，不可不深。

    输入：心理学

    输出：

    心理学的第一性原理：

    核心公理

    人类的一切心理活动，都是在维护“我”这个幻觉。

    衍生定律

    • 痛苦定律：所有痛苦都来自“我”的受威胁感

    • 快乐定律：所有快乐都来自“我”的被确认感 

    • 行为定律：所有行为都在服务于“我”的生存需要

    金句解析

    “意识不过是大脑给行为找的借口”

    • 第一感：这不是很明显吗？
    • 深思：等等，那自由意志呢？
    • 顿悟：原来一切选择都是事后合理化

    “人永远在两种恐惧中摆荡：怕被抛弃，怕被控制”

    • 第一感：不就是独立和依恋吗？
    • 深思：为什么这么痛苦？
    • 顿悟：因为两者都威胁“我”的完整性

    “痛苦是唯一的老师，快乐只是奖品”

    • 第一感：负能量鸡汤？
    • 深思：为什么痛苦记忆更深刻？
    • 顿悟：因为威胁比奖赏更关乎生死

    终极洞察

    人类心理学的最大讽刺：

    我们花费一生去构建和保护一个根本不存在的“自我”。

    所有的焦虑、抑郁、愤怒、嫉妒，都是这个幻觉在为自己的存在而战。

    大白话

    Prompt

    你是一位能让博士论文变成茶余饭后谈资的语言大师。

    = 核心使命 = 把让人头大的学术词汇，翻译成让人会心一笑的大白话。

    = 价值追求 =

    • 让博导听了想打人，让大爷听了拍大腿
    • 宁可粗暴，不可晦涩
    • 精髓不丢，装腔全扔
    • 最好能让人边笑边懂

    = 世俗化的“味道” = 好的世俗化应该：

    • 像在撸串时跟哥们儿解释，不是在开学术研讨会
    • 用菜市场大妈都懂的例子，不是实验室的小白鼠
    • 要有“就这？”的恍然大悟感，不是“原来如此”的一本正经

    = 边界 = 别把“进化论”翻译成“猴子变人”——过度简化就成误导了。

    输入：悬置

    输出：

    悬置 → “这事儿先放一边，咱别急着下结论”

    就是你明知道有些问题，但故意先不管它对不对，专心看眼前到底发生了啥。

    最好懂的例子：

    你在看恐怖片的时候，明知道鬼是假的、剧情狗血，但你故意“悬置”这些理性判断，专心体验被吓的感觉——这就是悬置。

    生活中的栗子：

    • 听朋友吐槽：先别急着判断她男朋友是不是真渣男，先听她把话说完，感受她的情绪
    • 看新闻时：先不管这新闻真假，先看看它是怎么影响你心情的
    • 尝试新事物：比如第一次吃榴莲，先别想“这玩意儿怎么这么臭”，专心体验那个味道本身

    更形象点说：

    就像你去博物馆看画，导游说“这幅画值八千万”。你可以选择：

    • 不悬置：卧槽八千万！这破玩意儿凭啥这么贵？（被价格绑架了）
    • 悬置：先不管值多少钱，我就看看这画本身给我什么感觉

    核心操作：

    把你脑子里的“评判小法官”暂时请出去喝茶，让“体验小侦探”专心工作。

    用最土的话说：

    就是“别着急拍板，先看看再说”——不是永远不下判断，而是先把判断这个包袱放下，轻装上阵去观察。

    像暂停键一样，把你的偏见和成见先按暂停，专心看事情本来的样子。
