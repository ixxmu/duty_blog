良心DLC被雪藏，打上补丁的Win10比11更强 by 电手

随着2024年 Windows 12 越来越近，Win11 接下来也不会有太多的变化了。
说起来相比 Win10 确实也多了不少新功能，只是稳定性以及能不能用上吧...
不过到底微软也还是行业大佬，做系统、做软件实力还是不容置疑的。

早在 Win95 时代就已发布的 PowerToys ，堪称业界良心。

后续不知道什么原因逐渐被微软雪藏，2019年才回归，到现在也更新了不少实用功能。

这个开源免费的工具集始终没有被内置 Windows 系统中，算得上微软的一大损失。
如果你还在怀疑微软出品的实力，下面就看看它有哪些强大且实用的功能吧。
 
01
   FancyZones   


Win11 增加了新的窗口分屏方式，这让许多 Win10用户眼馋。

而 FancyZones 的分屏相比 Win11 自带还要更加强大，众多实用的分屏模板并且还支持创建自定义布局。

强制全局、超多可自定义的设置、快捷键，让分屏的自由度提升了不止一个档次。

当然全默认按住 Shift 拖拽窗口基本已经可以满足需求了。

 
02
   实用鼠标工具   


高分辨率下很容易找不到鼠标？

只需要按两下 CTRL 键就能「查找我的鼠标」。

此外它还有许多可自定义的鼠标辅助功能，荧光笔、十字线…作为演示时十分方便。

 
03
   文本提取器   


Win+Shift+T 打开文本提取器，可以从屏幕上任意位置识别并复制文字。

默认支持中英文，还可以自行添加所需要的语言。

$Capability = Get-WindowsCapability -Online | Where-Object { $_.Name -Like 'Language.OCR*en-US*' }$Capability | Add-WindowsCapability -Online

04
    始终置顶    


和名字一样它可以让任意窗口置于顶层显示，免去来回切换生产力大大提升。

 
05
   颜色选择器   


按下 Win+Shift+C 可以启动颜色选择器，轻松采集复制该颜色值，看到到底有没有黄色。

 
06
  文件 Locksmith   


文件 Locksmith 可能是某些用户经常会用到的功能。

它可以直接在文件资源管理器中右键文件（夹）查看有哪些进程正在使用该文件。

有了它就不需要用大流氓来对付小流氓了。
 
07
   PowerToys Run   


类似macOS 的聚焦，按下 Alt+空格 打开 PowerToys Run 可以直接搜索软件、文件。

并且它还支持 Everything 增强搜索功能，有需要可以在Github 上找到内置Everything 的版本。
 
08
  Hosts 文件编辑器   


应对各种网络问题，修改 Hosts 文件可以说是必备操作了，不过每次都去翻目录实在太麻烦。

PowerToys 提供的 Hosts 文件编辑器能够以非常可视化的界面来添加、编辑Hosts 文件。


09
    结     语    


此外 PowerToys 还有很多实用工具，就不一一介绍了，大部分功能都能从名字知道其作用。

这个工具集总大小在最近一年也是从 100MB 更新到了 200MB ，又有许多实用新功能。

Github 项目地址：

https://github.com/microsoft/PowerToys
直接在 Microsoft Store 搜索下载也是可以的。

有趣的是最上面正是 Win10 + PowerToys > Win11 的评价。

可以说 PowerToys 中的任何一个功能拿出来都可以单独做个软件，微软将其集合在一起属实贴心。
只是到现在微软都似乎不想让 Windows 内置 PowerToys ，实在可惜啊。
