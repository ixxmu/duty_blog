折腾雷鸟电视

我其实非常不愿意去折腾一个电视，直到雷鸟通过强系统更新毁了我的电视！ 雷鸟电视手动强刷老版固件并删除自带应用，记录一下吧，毕竟是个大件，虽然和手机比起来刷的过程简短了些。 型号：625C 65寸 固件：从出厂到现在（2022年3月v105）的所有固件都是被官方强制升级，且没有回退和禁用开关，体验非常差！ 起因 雷鸟电视强制升级并且升级之后越来越卡顿，大几千买的一年多的电视就成个垃圾了，也不给个回退或者禁止升级的开关，真的对你们的行为感到非常恶心！ 真是白瞎了这硬件了，当然顺手搜了下TCL的电视都是这个鸟性，强制升级让电视报废是一贯操作了。 具体我也不想多罗列了 之前的帖子和B站视频已经挂上，帖子也会在互联网上慢慢扩散包括但不限于各大小社交网站！ （当然可能一些个大厂连某些隐私勾当踩着灰线的破事儿都能做出来，你们能干出来啥我都不惊讶了）

疫情防控--戴给中国人的另一个脚镣

疫情来袭的这几年，没有一人能够幸免，内地任何的疫情防控政策落到一个普通家庭的单位上都是一座山！ 现在对于大部分人来说口罩、核酸检测、隔离、防控这些名词已经变得如散在我们周围的空气一样无处不在，出门戴口罩、量体温、进门扫码、做核算我们每天都在经历，别人被隔离、区域被封闭、高架被劝返也 每天都在上演，在清零政策的原则下，疫情防控俨然已经变成了一项国策需要被长期坚持重点执行，任何一个出现COVID的地方都成了禁区，当你在地图上远远看着它的时候你会想，离我这么近或者还想着还远着呢， 但是当第二天醒来你变成了区域内的人的时候，一切来的是那么突然又那么毫无征兆！ 是的，我作为一个按部就班普通的上班族，2022年1月的某一天健康吗就突然由绿色变红了，在这之前本地所有疫情防控措施都积极配合从未也没有到过高风险地区，但是健康码就突然变红了， 只在数天前接到过自称流调的电话说我到过省内xx高风险区域，我能肯定的是最近几个月我都未来开过xx本地也从未去过其他任何高风险区域，变红码的第一时间我积极配合防疫政策并上报并配合了社区 执行居家隔离政策，但是我疑问在于我的健康码为什么变红了，知道具体原因也能让其他人避免问题，但是我打遍了所有公开途径的官方电话，没有一个部门能给出具体解释，只有这都是大数据上报的， 我们什么也做不了，并且当地省人民政府网上公开的24h疫情防控电话处于永远打不进电话状态； 同时居家隔离被要求每天不定时都要做核酸检测，包括口腔和鼻腔采样两次，做核酸检测配合防控政策真都是应该做的，但是14天内每天不定时两次采样是为什么， 同时家人也因为我的红码严重影响了个人的生活和工作，一家人因为家里的一个人在不知道什么原因的情况下被莫名其妙变成了红码就要承担停止工作、切断生活以及社会各方面的压力， 我不知道疫情政策还能持续多久也不知道发生在我身上的事情还能在多少人身上重演，也不知道病毒何时能被彻底消灭，如果防控政策需要持续进行下去， 我只希望能有针对疫情防控更细则的政策而不是黑匣子式的一刀切！

给 Github 添加 ssh 连接方式并排错

Github放弃了账号密码的验证方式，所以需要重新设定链接方式。 为何要有此一折腾？ Github 在08月15日彻底丢弃了账号密码验证的登录方式，之前的账号密码验证方式失效; 因为本身是在Rstudio上建目录，所以用的是普通用户，不是root账户，root账户下按照网上教程没有任何问题, 普通用户就是无法成功clone项目，因为必须用普通用户所以得搞定； 之前不愿意多看几步学习，这不来补课了。 标准添加方式暂时不表,可以参看文章后半部分 记录排错过程 按以上流程正常添加SSH之后，但是无法正常克隆项目到本地，使用 ssh -T git@github.

用 Github actions 编译任意路由器固件

给Glinet-MT-300Nv2编译最新xray固件 Update：10092021 因为Github限制资源的滥用，所以之前在线编辑config文件并编译的行为失效无法正常进行，请转到 大佬新仓库 继续拉取使用，新的逻辑为第一步仅生成对应设备的config文件，然后获取config内容，第二步将内容写进要编译的固件文件夹下面的 .config 文件，不要打开SSH在线编辑，重新编译即可。 分割线以下内容依旧有助于理解固件和在线编译的逻辑。 其实一直手里的Glinet-MT-300Nv2用着还算可以，直到v2ray分支诞生了xtls版的xray，新的配置方式虽然在手机或者电脑端用起来差别不大，但是在路由器端特别是这些个垃圾路由器不带AES解密的设备上就直接拉满了，所以就尝试在很久不更新固件的Glinet-MT-300Nv2上面尝试自己编译固件。 整个代码来自Github上的 别人项目,但是原作者删库跑路估计是不想折腾了，还是要感谢下大佬。这也提醒我们在Github上fork项目后pull项目前必须看一眼，万一大佬们改变主意失联了呢。
